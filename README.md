# Introduction
This project involves designing an interactive Power BI dashboard for Blinkit, India’s leading last-minute delivery app. The dashboard visualizes key metrics like total sales, number of items, average rates, fat content analysis, and outlet performance, providing actionable insights into Blinkit’s operations.

# Objectives
- Analyze Blinkit’s sales and operational data effectively.
- Visualize trends and patterns across outlets, item types, and locations
- Enable stakeholders to make informed decisions through interactive filters.


# Tools & Techniques
**Software:** Power BI

- Power BI – for data visualization and dashboard creation.
- Data Modeling & Cleaning – to structure and prepare data for insights.
- Interactive Filters – outlet type, size, item type, and location for dynamic analysis.

# Key Insights
- Identified top-performing outlets and item categories.
- Analyzed fat content and average rates by outlet to monitor product quality.
- Visualized sales trends and save-rate patterns across locations.

# Learning Outcomes
- Enhanced skills in Power BI dashboard creation and data visualization.
- Gained experience in data cleaning, modeling, and handling real-world datasets.
- Learned to present actionable business insights through interactive dashboards.

# Conclusion
The Blinkit Sales Dashboard provides a clear and interactive view of key business metrics, enabling data-driven decisions and operational improvements for Blinkit.
